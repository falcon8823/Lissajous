リサジュー図形のシミュレーション
==================================

オシロスコープで観測する様に，リサジュー図形をシミュレーションできます．

## 動作環境
* [Processing](http://processing.org)

## 使い方
ソースコード冒頭の「シミュレーション定数」を変えて実行するだけです．
